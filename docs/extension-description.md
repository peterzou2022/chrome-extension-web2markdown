# AI Page Summarizer – Store / Long Description

Use this text for the Chrome Web Store “Detailed description” or other long-form copy.

---

AI Page Summarizer is a lightweight Chrome extension that helps you quickly turn any webpage into a structured knowledge note.

With one click, the extension extracts the main content of the page, uses AI to generate a concise summary, and saves the result as a Markdown document to your local knowledge base.

It is designed for people who want to build a personal knowledge system while browsing the web.

## Key Features

1. **One-Click AI Summaries**  
   Instantly summarize any webpage using AI. Extract the key ideas, main points, and insights without reading the entire article.

2. **Clean Article Extraction**  
   Automatically removes ads, navigation, and clutter to capture the core content of the page.

3. **Markdown Export**  
   Generate structured Markdown notes that are easy to store, edit, and manage.

4. **Customizable Templates**  
   Define your own Markdown format for summaries, including sections like TLDR, Key Points, Insights, Quotes, and Notes.

5. **Local Knowledge Base Integration**  
   Save summaries directly to your local folder, making it easy to build a personal knowledge library.

6. **Flexible AI Configuration**  
   Use your own AI API key and choose the model you prefer.

## Example Output

Each summarized page is saved as a Markdown file:

```markdown
# Article Title

Source: https://example.com
Date: 2026-03-09

## TLDR
Short summary of the article.

## Key Points
- Key idea 1
- Key idea 2
- Key idea 3

## Insights
Important insights extracted from the content.

## Quotes
Notable quotes from the article.

## Notes
Your own thoughts and comments.
```
